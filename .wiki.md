# Project Summary
The C-Level AI Agent Service, now rebranded as "C-Level AI," is an innovative solution aimed at corporate executives, particularly those aged 50 to 70. It provides AI-driven insights and recommendations to enhance decision-making and operational efficiency. This service streamlines executive workflows, delivering critical data analysis that leads to improved business outcomes. The product features a user-friendly interface tailored for senior executives, enabling them to leverage advanced technology for strategic planning and operational excellence.

# Project Module Description
The project comprises several functional modules:
- **Core Principles**: Guidelines for AI interaction emphasizing verification and incremental development.
- **Development Guidelines**: Specifications for TypeScript usage, coding standards, and component design.
- **AI Collaboration**: Framework for AI agents' interactions and collaborations within the system.
- **Tasks and Milestones**: A structured roadmap for development, including task assignments and progress tracking.
- **Design System**: A comprehensive design framework ensuring accessibility and user-friendliness tailored for senior executives.

# File Description Inventory
- **src/app/**: Contains the main application logic and routing.
- **src/components/**: Houses all React components utilized in the application.
- **src/lib/**: Contains utility functions, API handlers, and business logic.
- **src/styles/**: Contains CSS and design system styles.
- **src/pages/**: Contains page components for different routes (e.g., Index, Login, Dashboard, Report).
- **src/types/**: TypeScript definitions for type safety across the application.
- **public/**: Static assets such as images and fonts.
- **.env.local**: Configuration for environment-specific variables.

# Technology Stack
- **Frontend**: Next.js 14, TypeScript, React
- **Styling**: Tailwind CSS, shadcn/ui
- **Backend**: Supabase (PostgreSQL + Auth + Storage)
- **AI/LLM**: OpenAI GPT-4, Supabase Vector (pgvector)

# Usage
1. **Install Dependencies**:
   ```
   npm install
   ```
2. **Run Development Server**:
   ```
   npm run dev
   ```
3. **Build for Production**:
   ```
   npm run build
   ```
